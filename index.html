📖 How Price Schedules Work
1️⃣ Save original prices
Set effective date → Jan 1 → Save as Scheduled Prices. This locks current prices for Jan onwards.
2️⃣ Apply new prices
Use Bulk Adjust % or edit prices manually → Save Prices & Weights.
3️⃣ Save new prices
Set effective date → April 1 → Save as Scheduled Prices. April onwards uses new prices.
✅ Result
Jan–Mar use old prices. April onwards uses new prices. All data untouched.
